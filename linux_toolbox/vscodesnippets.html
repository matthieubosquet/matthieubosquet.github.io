--- title: html layout: default --- To access user snippets, go to Code > Preferences > User Snippets and add the content to the relevant language. The "prefix" field indicates the sequence of characters allowing to trigger the snippet in vscode. The "body" field defines the snippet. The "description" is shown in the editor when the snippet is active.
