Install

Base

Install node, npm (node package manager) and git

{% highlight bash %} $ apt-get install node npm git {% endhighlight %}

Install bower globally (-g to be able to run the command from terminal)

{% highlight bash %} $ npm install -g bower {% endhighlight %}

Initialise my project with bower to handle dependencies (bower.json config file)

{% highlight bash %} $ bower init {% endhighlight %}

Install polymer for this project and save this dependency in config

{% highlight bash %} $ bower install Polymer/polymer --save {% endhighlight %}

Dependencies are fetched in the bower_components folder it can be deleted and re-installed later with

{% highlight bash %} $ bower install {% endhighlight %}

Extras

Polymer core elements

{% highlight bash %} $ bower install Polymer/core-elements {% endhighlight %}

Polymer paper elements

{% highlight bash %} $ bower install Polymer/paper-elements {% endhighlight %}

Update

{% highlight bash %} $ bower update {% endhighlight %}
